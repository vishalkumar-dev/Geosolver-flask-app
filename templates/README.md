# GeoSolver Pro

A lightweight and responsive Geometry Calculator Web App built using Python Flask.

It allows users to perform coordinate geometry calculations instantly with a clean interface and visual graph output.

## Features

- Real-time Distance Calculation between two points
- Midpoint Calculation
- Slope Calculation
- Calculate all results at once
- Coordinate Plane Graph Plotting
- Export Result as PDF
- Recent Calculation History
- Clean and Responsive Dark UI

## Tech Stack

- Backend: Python, Flask
- Frontend: HTML, CSS
- Graph Visualization: Plotly
- PDF Export: ReportLab

## How to Run Locally

1. Clone the repository

```bash
git clone <your-repo-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open browser

```text
http://127.0.0.1:5000
```

## Project Goal

This project was built to practice Python Flask development, geometry calculations, graph visualization, and backend integration.
